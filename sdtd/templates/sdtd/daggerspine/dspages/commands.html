Commands
Green commands = CBSM commands
Blue = CBSM and OSM
Yellow commands = OSM commands
CBSM-Cloud Based Server Manager: Commands are subject to lose connection and not be available.
OSM-On Server Manager: This is a manager sitting on the server with the game and cannot lose connection

Getting Started

/sethome
Command Details: Sets your home at the current location.
Usage Example: /sethome
Fee: 5 zCoins
/home
Command Details: Teleports you to your home.
Usage Example: /homeFee: 1 zCoins
Cooldown: 10 Minutes


Economy

Players earn 1 zCoin per zombie killed, and 0.20 zCoins per minute played.
/shop
Command Details: This will display a list of non-grouped items within the server shop.
Usage Example: /shop
Related Command: /shop list (This will display a list of grouped items within the server shop.Usage Example: /shop food)
/buy
Command Details: This is used to purchase the item from the server shop. The item is specified by ID, as found from the /shop command.
When you purchase an item, it will appear at your feet. Press E to pick it up
Usage Example: /buy 52605
/wallet
Command Details: Displays your wallet balance.
Usage Example: /wallet
/pay [#] zCoins to [name of player]
Command Details: This will send the specified amount to the provided player.
Usage Example: /pay 50 zCoins to zombiehunter17
/money
Command Details: Displays a list of the top 10 wealthiest players.
Usage Example: /money


Fast Travel

/zgate create [name]
Command Details: Creates a zGate with the provided name at your current location.
Usage Example: /zgate create mySpot
Creation Fee: 25 zCoins
Max zGates / Player: 3
/zgate visit [name]
Command Details: Teleports you to the zGate that you specified. You must be the owner of the zGate, or it must be set to public.
Usage Example: /zgate visit Hotel
Usage Fee: 5 zCoins
Usage Delay: 10 Minutes
/zgate toggle [name]
Command Details: Changes the privacy status of the zGate from Private to Public. And Public, back to Private.
Public zGates can be visited by anyone!
Usage Example: /zgate toggle mySpot
/zgate delete [name]
Command Details: Deletes the zGate. Can only be done by the zGate owner, and server admin.
Usage Example: /zgate delete mySpot

Daggerspine Specific Commands

/Bounty List
Command Details: Displays a list of all current bounties
Cooldown: 0
Fee: 0
/Bounty Set <#> zcoins on {name}
Command Details: Creates a new bounty worth the specified <#> of zCoins on the provided {name} -
For example /bounty set 15 zcoins on AncientLotus
Cooldown: 0
Fee: Bounty Amount
/Bounty Unset {name}
Command Details: Removes all of your bounties you have placed on the provided player {name}, and returns zCoin bounty to your wallet.
Cooldown: 0
Fee: 0
